Advertisment Loader JS

The code displays an advertisement on the Transport Topics News website once per minute. The advertisement disappears after the user refreshes the page and is in compliance with Google Ads regulations. This code uses the localStorage object to store the time that the advertisement was last displayed. If the advertisement has not been displayed recently, it will be displayed. The advertisement will then be hidden after 1 minute.
